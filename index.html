C'est quoi le CSS ?


CSS (cascading style sheet) est un langage de programmation informatique qui sert à mettre en forme la structure d'un site en HTML. Il consite donc à apporter des modifications sur un site, en prenant en compte la modification des bordures, la couleur des textes, la couleur de fond, la taille de l'écriture etc. En un mot il permet de rendre un assez esthétique et agréable à la vue.

Les versions du CSS

Le CSS a connu une évolution depuis sa cration en 1996.

  • CSS 1: Ceete première version fait son apparition en 1996 et pose les bases du langage. Cette base permettait de modifier les pages web e prenant en compte les couleurs, les marges et les polices de caractère etc.

  • CSS 2 : La version 2 fait son apparition dans les années 1999. Les utilisateurs peuvent désormais utiliser les techniques de positionnement très précis. C'est-à-dire, afficher les éléments où l'on le sur la page.

  • CSS 3 : C'est la toute dernière version tant attendue de la part des utilisateurs, elle permet d'apporterdes modifications telles que les bordues arrondies, les ombres, les dégradés etc.
